## pip 인식 안될때 해결 방안
 1. 환경 변수 (Windows): pip 명령어가 인식되지 않으면 파이썬 Scripts 폴더를 PATH 환경 변수에 추가
 2. 파이썬 설치 경로 확인
    - C:\Users\{사용자PC명}\AppData\Local\Programs\Python\{Python버전폴더}
    - EX) C:\Users\samso\AppData\Local\Programs\Python\Python314
 3. WINDOWS 11 버전 기준
   - 설정 > 시스템 > 정보 > 고급 시스템 설정 > 시스템 속성 창(고급탭) > 환경변수 > 시스템 변수 > PATH 선택 > 새로만들기 >
     파이썬 설치 경로 추가
     EX> C:\Users\samso\AppData\Local\Programs\Python\Python314\
         C:\Users\samso\AppData\Local\Programs\Python\Python314\Scripts\
