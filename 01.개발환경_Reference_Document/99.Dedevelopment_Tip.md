 ## VSCode 단축키 모음

 - 새로고침      : F5
 - 저장          : Windows: Ctrl + S / macOS: command + S
 - 전체선택      : Windows: Ctrl + A / macOS: command + A
 - 잘라내기      : Windows: Ctrl + X / macOS: command + X
 - 콘솔창 줄바꿈 : shift + enter
 - 코드정렬      : Windows: Ctrl + Alt + L / macOS: option + command + L
 - 들여쓰기      : Tab
 - 들여쓰기 취소 : Shift + Tab
 - 주석          : Windows: Ctrl + / / macOS: command + /
